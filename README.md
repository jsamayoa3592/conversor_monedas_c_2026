# Conversor de Monedas en C

## Nombre del proyecto
Conversor de monedas entre quetzales, dólares y euros.

## Descripción
Este proyecto consiste en una aplicación desarrollada en el lenguaje de programación C que permite realizar conversiones entre quetzales, dólares estadounidenses y euros mediante un menú interactivo en la terminal.

## Prerrequisitos
Para ejecutar el proyecto es necesario contar con:

- Sistema operativo Linux o Ubuntu
- Compilador GCC
- Terminal de comandos
- Editor de código (Visual Studio Code)

## Pasos para ejecutar el proyecto

1. Abrir la terminal.
2. Ubicarse en la carpeta del proyecto.
3. Compilar el programa con el siguiente comando:gcc conversor.c -o conversor
4. Ejecutar el programa: ./conversor
5. Seleccionar la opción de conversión deseada e ingresar la cantidad.
